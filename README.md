# Caesar Zipher
Программа для работы с принтером VideoJet по протоколу Zipher. Используется для динамической
печати кода маркировки системы "Честный ЗНАК" на этикетке. 

## Использование
1. Подготовить txt-файл с КМ. Один КМ на строчку.
2. Запустить программу и нажать "Загрузить коды", а затем выбрать файл с КМ.
3. Нажать кнопку "#" и убедиться в правильности настроек.
4. Нажать кнопку "Не подключен" и дождаться подключения к принтеру.
5. Нажать кнопку "Остановлен", чтобы запустить работу программы.

## Требования к макету этикетки
Макет подготавливается в программе CLARISOFT.  
В макете этикетки должны присутствовать два поля:  
- Поле Datamatrix
- Поле с GTIN (его можно вынести за границы этикетки, чтобы оно не печаталось)  

Поле Datamatrix будет использоваться для динамической печати кода маркировки ЧЗ, поэтому тип поля
необходимо выставить "User Entered Text". Кроме того, в силу особенностей макетов в программе
Clarisoft, необходимо вручную выставить длину поля.  
Тип поля с GTIN следует установить "Fixed Text" и в нем указать GTIN продукции. Это поле нужно для
того, чтобы при работе программы проверялось, что загруженные КМ относятся к той же продукции,
что и макет. Если это поле будет отсутствовать или оно будет незаполнено, то проверка происходить
не будет.  
Двум перечисленным полям в макете следует дать осмысленное наименование, а затем указать его в
настройках программы.
